smali文件动态调试出现的问题

关于动态调试smali文件出现的一些问题

首先把smali文件夹导出后一定要放在一个文件夹里面并且将smali文件改名为src

检查xml文件是不是有允许调试的语句

关于android device monitor在Android SDK安装文件夹tools文件夹下monitor.bat双击即可打开

输入命令adb shell -D -n hfdcxy.com.myapplication/hfdcxy.com.myapplication.MainActivity只能使用这个命令，其他命令搜索不到

android device monitor 查看online端口，查看完就关闭，否则后面会转发端口失败

打开命令行窗口 输入命令 adb forward tcp:8700 jdwp:19509 转发8700 端口 (这里 jdwp是自己Android Device Monitor中要调试app的Online值 )

后面重新打开Android studio调试文件不能下断点，后来发现重新调试文件的时候没有安装ideasmali，要重新在安装上才能下断点
